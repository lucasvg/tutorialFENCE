Conhecendo o FENCE

Acessando o servidor

Para acessar o servidor via ssh: 

ssh atglance@lxplus.cern.ch
Uma instância do FENCE que podemos usar para aprender está no diretório ~/www/stage-lucas/ 
cd ~/www/stage-lucas/


Dica:

Um editor usado por linha de comando bem simples é o Nano. Por exemplo, para abrir um arquivo, use "nano configuration.json". (Pode-se fazer o download e editar os arquivos localmente. Porém, por facilidade de ver o resultado no navegador, é preferível alterar remotamente via ssh...)

Organização dos diretórios

Dentro deste diretório haverá duas pastas, fence e atlas.
Na pasta fence, estão todos os arquivos necessários para se ter um sistema FENCE rodando.
Na pasta atlas, os arquivos necessários para fazer a integração com os dados do experimento Atlas. (No lugar da mesma poderia ser usado uma extensão para o sistema Alice, LHCb...)

Pode-se visualizar o resultado via web, através das seguintes URLs:
https://atglance.web.cern.ch/atglance/stage-lucas/atlas - corresponde ao diretório ~/www/stage-lucas/atlas/
https://atglance.web.cern.ch/atglance/stage-lucas/fence - corresponde ao diretório ~/www/stage-lucas/fence/


Alguns arquivos importantes

~/www/stage-lucas/fence/configuration.json
Define variáveis utilizadas em vários outros arquivos. Neste arquivo é "montado o sistema em si", quais diretórios existirão, quem poderá acessar cada nivel... Se um diretório não contiver o seu arquivo configuration.json, o arquivo configuration.json do diretório acima será usado. Por exemplo, se acessarmos https://atglance.web.cern.ch/atglance/stage-lucas/atlas/appointment/, terá precedência o arquivo ~/www/stage-lucas/atlas/appointment/configuration.json. Se este não existir, será usado o arquivo ~/www/stage-lucas/atlas/configuration.json

~/www/stage-lucas/fence/environments.json
Define os ambientes de desenvolvimento.

~/www/stage-lucas/atlas/.htacess
Além de outras configurações, como o modo de autenticação de usuário e conexão com o BD, define qual ambiente de desenvolvimento está em operação.

~/www/stage-lucas/fence/menu.json
Através deste arquivo é possível alterar os links que são exibidos na página, neste caso, em https://atglance.web.cern.ch/atglance/stage-lucas/fence/.
